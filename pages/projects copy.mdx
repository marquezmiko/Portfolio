## Atelier
## E-Commerce front end with scaled microservices on various modules
- Created React-based components which displayed an overview of the selected product and front-end interaction
- Implemented an API microservice for the products that achieved up to 1,500 RPS with <0.1% error rate and <18ms latency
- Refactored PostgreSQL queries to increase efficiency on millions of records, in one case improving from a 5 second avg to 8ms
- Utilized NGINX load balancer with Redis caching to increase latency performance by 99.1%
- Stress tested using Loader.io and NewRelic, to assess hardware limitations and drive data-based optimization decisions
